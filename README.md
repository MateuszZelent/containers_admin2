# Containers Admin 2

## Uruchomienie z Docker

### Wymagania
- Docker
- Docker Compose

### Instalacja i uruchomienie
1. Sklonuj repozytorium
2. Przejdź do katalogu projektu
3. Uruchom aplikację poleceniem:
   ```
   ./start.sh
   ```
   lub
   ```
   docker-compose up -d
   ```
